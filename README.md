# DOTA
Age of Empires 3 (2005) custom map inspired by Dota 2. 

## Installation
Download dota.xs and dota.xml files, and copy them into the Documents/My Games/Age of Empires 3/RM directory. Start the game (the original 2005 game, it's does not work for AoE DE, TWC or TAD). Select Multiplayer -> LAN. Select "Host Game" and check "Custom Maps". Pick " dota" from the dropdown list.
